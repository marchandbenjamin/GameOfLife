This game of life application include a 9x9 red cells table.
A red cell is a dead cell, once your click on a red cell it becomes a green cell.
A green cell is a living cell.

There is also a button "Generation suivante". 
This buttons changes the cell table according to the following rules :
If a living cell has more than three neighbours, it dies.
If a living cell has less than two neighbours, it dies.
If a dead cell has exactly three neighbours, it becomes a living cell.
